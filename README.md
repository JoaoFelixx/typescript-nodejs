# typescript-nodejs

## contents / Conteúdo

Nesse projeto eu estava usando **Node JS** para aprender o typescript e sinceramente gostei bastante de usar ele.

* Primeiramente pelo fato de não precisar aprender uma nova linguagem de programação com tipagem forte. Ex: Java, C#, Cobol tec.
* Segundamente ajuda na hora de desenvolver filtragens, redeclaração de variáveis, refatoração de código, expansão de código, etc.
* Terceiramente achei a sintaxe extremamente elegante e estilosa.

## Tá, mas afinal vale a pena aprender TypeScript ?

Eu sinceramente tambem achava que não fazia muito sentido estudar sobre, mas parando para
aprender um pouco eu notei algumas características interessantes como: **interfaces, types, tipagem,** etc.
Ou seja, aprenda, se seu foco for `Javascript` aprenda ! vale bastante a pena.
